# IMX219-Driver-For-Jetson

- IMX219摄像头[说明链接](https://www.waveshare.net/wiki/IMX219-77_Camera)

## 测试相机

将IMX219通过排线连接到JetsonNano上后，在终端输入下命令即可打开摄像头预览拍摄效果

```bash
$ nvgstcapture
```
